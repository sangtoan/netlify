Nhận xét nào dưới đây chắc chắn \textbf{sai}? \choice {$\overrightarrow{u} = \overrightarrow{0}$} {$\overrightarrow{v} = \overrightarrow{0}$} {\True $|\overrightarrow{u}\cdot \overrightarrow{v}| < |\overrightarrow{u}|\cdot |\overrightarrow{v}|$} {$\overrightarrow{u}\parallel \overrightarrow{v}$} \loigiai{ Với $\overrightarrow{u}=\overrightarrow{0}$ hoặc $\overrightarrow{v}=\overrightarrow{0}$ hoặc $\overrightarrow{u}\parallel \overrightarrow{v}$ thì $\left[ \overrightarrow{u},\overrightarrow{v} \right]=\overrightarrow{0}$ nên khẳng định chắc chắn \textbf{sai} là $|\overrightarrow{u}\cdot \overrightarrow{v}| < |\overrightarrow{u}|\cdot |\overrightarrow{v}|$.\\ Ta có thể chỉ ra ví dụ bằng cách lấy $\overrightarrow{u}=\overrightarrow{v}=(1;1;1)$ thì $\left[ \overrightarrow{u},\overrightarrow{v} \right]=\overrightarrow{0}$ nhưng \[ |\overrightarrow{u}\cdot \overrightarrow{v}|=3=\sqrt{3}\cdot\sqrt{3}=|\overrightarrow{u}|\cdot |\overrightarrow{v}|. \] } \end{ex} \begin{ex}%% %%[2H3B2-1] %Câu 36: [TDM31] Trong không gian với hệ tọa độ $O x y z$, cho bốn điểm $A(2 ; 1 ; 1),$ $ B(-2 ; 2 ; 1),$ $ C(0 ; 3 ; 1)$, $D(3 ; 0 ; 0)$. Khoảng cách giữa hai đường thẳng $A B$ và $C D$ bằng \choice { $\dfrac{3}{\sqrt{15}}$} { $\dfrac{1}{2}$} { $\dfrac{\sqrt{2}}{6}$} {\True $\dfrac{11}{\sqrt{17}}$} \loigiai{ Đường thẳng $AB$ đi qua $A(2;-2;0)$ và có véc-tơ chỉ phương $\vec{AB}=(1;2;2)$;\\ Đường thẳng $CD$ đi qua $C(-1;1;2)$ và có véc-tơ chỉ phương $\vec{CD}=(-1;0;1)$;\\ Ta có $\vec{AC}=(-3;3;2)$ và $[\vec{AB},\vec{CD}]=(2;-3;2)\Rightarrow [\vec{AB},\vec{CD}]\cdot \vec{AC} =-11.$\\ Vậy d$(AB,CD)=\dfrac{\left|[\vec{AB},\vec{CD}]\cdot \vec{AC}\right|}{\left|[\vec{AB},\vec{CD}]\right|}=\dfrac{11}{\sqrt{17}}.$ } \end{ex}
